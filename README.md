# Film Flu
Film Flu is a project that show the capacity of Flutter and flutter_bloc in different enviroment using BLoc Architecture Pattern and Clean Architecture design.

## PACKAGES USED:
    - Riverpod
    - Freezed
    - Dio
    - Envied Generator
    - Intl
